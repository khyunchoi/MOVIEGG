## 주요 기능 목록💪

- 네이버, 카카오, 구글, 깃헙 계정과 연동한 소셜 로그인 기능
- 이메일 인증을 통한 비밀번호 찾기 기능
- 영화 및 게시판 검색 기능
- 유저 레벨에 따른 권한 관리 기능
- 영화진흥위원회 API, TMDB API, 크롤링 등을 이용한 영화 추천 기능
- 캐릭터 고르기 테스트를 통한 영화 추천 기능
